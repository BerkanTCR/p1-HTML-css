Onderzoeken

Waarom noem je de eerste HTML-pagina; index.html?

Het dient als de startpagina. Webservers openen index.html wanneer een gerbruiker een directory opvraagt zonder een bestandsnaam.

Met welke HTML versie werken we nu?

We werken met HTML 5.

Met welke CSS versie werken we nu?

We werken met CSS3.

Hoe kan het dan een website er net iets anders uit ziet in een andere browser? (Edge, Chrome, Firefox, Opera)

Browsers lezen de code van websites op verschillende manieren. Er zijn kleine verschillen tussen moderne browsers. Dat komt door de Flexbox en grid layouts. De CSS-layoutsysteem implementeert fijne details wat anders.

Welke browser is het populairst en hoeveel procent RAM gebruikt deze browser?

Google Chrome, Google Chrome gebruikt tot 10% van je RAM. Dat is omdat het applicatie zo snel mogelijk wilt runnen. Een andere reden is cahcing.
